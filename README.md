# 情报搜索助手

一个基于 Flask 的小型网页工具，用于按关键字组合搜索情报，并支持时间范围过滤和离线 AI 结果提示。核心爬虫逻辑来源于提供的 Selenium 示例。

## 特性
- 关键字随机组合：输入关键词后，一键随机生成或洗牌组合。
- 时间范围预设：最近一天、三天、一个月或自定义时间段。
- 搜索结果展示：标题、作者、时间、媒体中英文名、正文摘要和加载耗时。
- AI 分析接口：在当前搜索结果中勾选条目后，提交问题可获得基于上下文的提示性回答（离线规则模拟）。

## 本地运行
1. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```
2. 启动服务
   ```bash
   python app.py
   ```
3. 浏览器访问 `http://localhost:5000`。

> 说明：爬虫依赖本地可用的 Chrome 与 ChromeDriver（Selenium Manager 自动匹配）。在无图形界面的环境下运行可能失败。
